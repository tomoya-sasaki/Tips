# Edge
## Line Type
See [here](1)

```
\edge[1, 1-1, 1-2](2)(3)
% 1: line type (i.e.dashed)
% 1-1: Line direction (<->, ->, <-)
% 1-2: shorten the lenth ("shorten >= -5pt", "shorten >=< -10pt")
% 2: origin of the line
% 3: destination of the line
```
